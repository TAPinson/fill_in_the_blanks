


Requirements:
If the following are not already installed, you can use the commands listed below to get them running on your machine.
- Python 2.7 or later
    -sudo apt-get update
    -From the terminal: sudo apt-get install Python
- GIT
    -sudo apt-get install update
    -From  the terminal: sudo apt-get install GIT



For Installation and Program Up:
- Open a terminal with the key combination ctrl + alt + t
- Clone directory using command "git clone https://github.com/TAPinson/fill_in_the_blanks.git"
- Change directory to the newly cloned directory using "cd fill_in_the_blanks"
- Run the program with the command "python fill_in_the_blanks.py"
